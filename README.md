# Genshin Characters

> 원신 플레이어블 캐릭터 분류기.
>
> 무과금, 소과금 유저의 선택과 집중을 도와주기 위한...



## 데이터

> 주로 나무위키 참조.
>
> 손으로 직접 입력.



## 캐릭터 분류

* 원소
* 무기
* 지역
* 성별
* 등급
* 돌파 스탯



## 추가하고 싶은 내용

### 분류 기준

* 돌파 재료 1, 2, 3.



### 그 외

* 각 속성별 분포 페이지.
* 모바일 웹 브라우저 환경 고려.
